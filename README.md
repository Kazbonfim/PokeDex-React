# PokéDex - React
Projeto simples, e rápido, pra aprender alguns conceitos de JSX e de React; fiz em uma noite, então não espere nada mirabolante.
Usando AXIOS para facilitar requisições, a aplicação exibe (e apenas exibe) os 251 Pokémon, com dados como: ID, nome, tipos, e uma image simples de seus respectivos Sprites (padrão).

## Como usar o projeto?
Bem, caso queira, dê um git clone, vá até a pasta onde os arquivos estão - se precisar, extraia - e faça o seguinte comando (lembre-se de direcionar seu CLI pra raíz da pasta):

```
npm install
npm run dev
```
Bem simples, não? Agora, só abrir seu navegador, e usar a porta padrão que o Vite direciona. Não fiz refinamentos, é realmente um projeto pensado pra ser curto, e entender alguns conceitos, como os estados...
Se quiser contribuir com alguma coisa, já sabe, basta marcar em ISSUES ou simplesmente fazer uma PR, até mais!!
